# Реестр экранов приложения «Вертикаль»

Данный документ содержит перечень всех экранов мобильного приложения, необходимых для реализации функциональных требований MVP и post-MVP.

| ID | Название экрана | Описание | Связанные требования | Приоритет |
|----|-----------------|-----------|---------------------|------------|
| SCR-01 | Экран регистрации | Регистрация нового пользователя (email, пароль, подтверждение 18+) | FR-001, FR-002, UC-001, US-001 | Must |
| SCR-02 | Экран авторизации | Вход в систему для зарегистрированных пользователей | FR-001, UC-001 | Must |
| SCR-03 | Экран принятия оферты | Принятие правил безопасности и оферты при первой записи | FR-003, UC-003, US-002 | Must |
| SCR-04 | Главный экран / Расписание | Список доступных слотов на 7 дней, фильтры поиска | FR-005, FR-006, FR-007, FR-008, UC-002, US-003 | Must |
| SCR-05 | Экран бронирования | Выбор количества мест, выбор снаряжения, просмотр итоговой стоимости и правил отмены | FR-010, FR-014, FR-016, FR-017, FR-018, FR-019, UC-003, US-004, US-005, US-006 | Must |
| SCR-06 | Мои бронирования | Список активных и отмененных записей клиента | FR-011, FR-022, FR-023, FR-025, UC-004, US-007 | Must |
| SCR-07 | Детали бронирования | Подробная информация о конкретной записи, кнопка отмены | FR-022, FR-023, UC-004, US-007 | Must |
| SCR-08 | Инфо-центр / Правила | Документ с правилами посещения (адрес, режим, что взять) | FR-004, US-010 | Must |
| SCR-09 | Настройки уведомлений | Настройка времени push-напоминаний о тренировках | FR-028, US-008 | Must |
| SCR-10 | Оценка инструктора | Форма оценки (1-5) и текстовый отзыв (post-MVP) | FR-031, FR-032, UC-006, US-011 | Could |
