# Постановка на дизайн: SCR-08 Инфо-центр / Правила

## Описание
Информационный экран, содержащий справочную информацию о посещении скалодрома.

## Цель
Обеспечить клиента всей необходимой информацией для подготовки к визиту (адрес, режим работы, рекомендации).

## Функциональные элементы
- **Блок «Адрес и контакты»**: Текстовая информация с адресом, возможностью скопировать адрес или открыть его в картах.
- **Блок «Режим работы»**: Расписание работы скалодрома по дням недели.
- **Блок «Что взять с собой»**: Список рекомендаций по одежде и обуви.
- **Блок «Правила посещения»**: Основные правила поведения и безопасности.

## Требования к UX/UI
- Структурированный текст с использованием заголовков и списков для легкого сканирования.
- Использование иконок для визуального разделения блоков (например, иконка карты для адреса, иконка часов для режима работы).
- Чистый, текстоцентричный дизайн.

## Связанные требования
- FR-004, US-010
