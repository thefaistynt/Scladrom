# Постановка на дизайн: SCR-04 Главный экран / Расписание

## Описание
Центральный экран приложения, где клиент просматривает доступные слоты для записи на тренировки.

## Цель
Позволить пользователю быстро найти подходящий слот по дате, времени, инструктору или формату.

## Функциональные элементы
- **Календарь/Выбор даты**: Горизонтальный скролл дат на 7 дней вперед по умолчанию.
- **Блок фильтров**:
    - Выбор формата тренировки.
    - Выбор инструктора.
    - Время суток (Утро/День/Вечер).
    - Переключатель «Только свободные места».
- **Список слотов**: Карточки тренировок, содержащие:
    - Время начала и окончания.
    - Название формата.
    - Имя инструктора.
    - Количество оставшихся мест (например, «3/10 свободно»).
- **Empty State**: Сообщение «Пока нет доступных тренировок», если по выбранным фильтрам ничего не найдено.
- **Навигация**: Переход в «Мои бронирования» (SCR-06) и «Инфо-центр» (SCR-08).

## Требования к UX/UI
- Интуитивно понятная навигация по датам.
- Быстрый доступ к фильтрам (возможно, через иконку фильтрации).
- Визуальное выделение текущего дня.
- Карточки слотов должны быть кликабельными для перехода к бронированию (SCR-05).

## Связанные требования
