# SCR-04 Расписание тренировок

**ID:** SCR-04  
**Тип:** Экран  
**Домен:** Расписание  
**Приоритет:** Critical  
**Статус:** Черновик  
**Зона авторизации:** АЗ  

---

## Обзор

Главный экран приложения, где клиент просматривает доступные слоты на ближайшие 7 дней, применяет фильтры и выбирает тренировку для бронирования.

### User Story

> Как клиент, я хочу видеть список доступных тренировок с фильтрами, чтобы быстро выбрать подходящий слот.

### Бизнес-ценность

- сокращает время поиска подходящей тренировки;
- помогает клиенту принять решение без звонка или переписки;
- уменьшает число конфликтов между клиентами и персоналом.

## Входные данные

| Название | Тип | Описание |
|---|---|---|
| selectedDateRange | date range | Диапазон дат для отображения |
| filters | object | Формат, инструктор, время суток, только свободные |
| slots | array | Список тренировочных слотов |

---

## Элементы экрана

- календарь/горизонтальный список дат;
- блок фильтров;
- список карточек слотов;
- empty state «Пока нет доступных тренировок»;
- навигация в мои бронирования и инфо-центр.

---

## Состояния экрана

| Состояние | Поведение |
|---|---|
| Loading | Показать skeleton или loader |
| Loaded | Показать список слотов |
| Empty | Показать empty state |
| Error | Показать ошибку с кнопкой «Обновить» |

---

## Действия пользователя

1. Выбрать дату из списка.
2. Применить фильтры.
3. Выбрать слот.
4. Перейти в мои записи или инфо-центр.

---

## Используемые запросы

### GET /slots

**Триггер:** Открытие экрана и изменение фильтров.

**Параметры:** fromDate, toDate, formatId, instructorId, timeOfDay, onlyAvailable.

**Обработка ответа:**

- success → показать список слотов;
- пустой список → empty state;
- ошибка сети → state error.

## Критерии приёмки

- Дано: клиент открыл экран, **Когда** он ещё не выбрал фильтры, **Тогда** система показывает слоты на 7 дней вперёд.
- Дано: по выбранным фильтрам нет слотов, **Когда** клиент применяет фильтры, **Тогда** показывается empty state.
- Дано: клиент выбрал слот, **Когда** он нажимает на карточку, **Тогда** открывается экран бронирования.
